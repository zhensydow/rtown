* Robot Town

Robot Town is a Sandbox Game used for me to exercise my develop skills and learn
the Löve Framework.

Robot Town is intended to be an 2D RPG.

** Brief Manual
 + Install Löve (https://love2d.org/)
 + Download/Clone rtown
 + Execute inside rtown folder:
#+BEGIN_EXAMPLE
love .
#+END_EXAMPLE
